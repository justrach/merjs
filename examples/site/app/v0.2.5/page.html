Added

Zig 0.16 support

The framework, CLI, and examples all now target Zig 0.16.0. The migration guide documents the breakage classes and the exact replacements.

Added

Edge-served installer

The install script now serves from Cloudflare Workers so the first contact point with merjs is also running on the edge stack it supports.

Changed

Modernized core APIs

Networking, IO, time, and collection setup all moved to the new Zig 0.16 idioms, reducing drift from upstream language evolution.
