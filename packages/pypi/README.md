# merlionjs

Next.js-style web framework in Zig — zero Node.js required.

## Installation

```bash
pip install merlionjs
```

## Usage

```bash
# Create new project
mer init my-app
cd my-app

# Start dev server
zig build serve

# Build for production
zig build prod
```

## Requirements

- Zig 0.15.1+
- Python 3.8+ (for this installer only)

## Platform Support

- macOS (Intel & Apple Silicon)
- Linux (x64 & ARM64)
- Windows (x64)
